# 자산 다운로드{#download-an-asset}

Adobe Experience Cloud 라이브러리에서 자산을 다운로드할 수 있습니다.

1. 자산을 클릭합니다.
1. 자산 옆에 **[!UICONTROL 있는 추가 옵션]** 메뉴 (줄임표) 를 클릭합니다.

   ![](assets/library_asset_options.png)

1. **[!UICONTROL 다운로드를]** 클릭하여 자산을 다운로드합니다.
