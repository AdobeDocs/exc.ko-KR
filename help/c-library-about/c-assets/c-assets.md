# 자산{#assets}

Adobe Experience Cloud 라이브러리의 다양한 툴 간에 에셋을 관리하여 손쉽게 세부 정보와 버전을 찾아 에셋에 대한 작업을 수행할 수 있습니다.

Experience Cloud 라이브러리에서 다음을 수행할 수 있습니다.

* 큰 아이콘 보기에서 자산을 미리 보거나 세부 사항 목록 보기에서 세부 사항을 볼 수 있습니다.
* 개별 자산에 대한 세부 사항을 봅니다.
* 에셋을 교체합니다.
* 자산의 이전 버전을 보고 되돌립니다.
* 자산 휴지통
* 이전에 삭제한 에셋을 복원합니다.
* 분산된 에셋을 영구적으로 삭제합니다.
